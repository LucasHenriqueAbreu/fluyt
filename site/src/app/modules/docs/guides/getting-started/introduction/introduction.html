Thank you for choosing Fluyt.

Fluyt is a multi-purpose, multi-layout Angular admin template featuring custom made Application and Page layouts, UI elements and UX best practices.

In this documentation, you will find starter guides and API documentation of the custom made Fluyt components.

Main libraries

Fluyt powered by 3 main libraries; Angular, Angular Material and TailwindCSS:

  • Angular

    It is the core of the Fluyt. This version only works with Angular.

  • Angular Material

    Provides the main UI components such as Buttons, Form fields, Tabs, Progress bars and more.

  • TailwindCSS

    It is the core of the style configuration and it provides utility classes for almost every CSS rule available.
